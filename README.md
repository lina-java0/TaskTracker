Backend-приложение для управления текущими задачами.

На текущем этапе реализована базовая структура приложения:
настроена база данных для хранения задач
реализован REST API
доступ к функциональности осуществляется через Postman

Приложение позволяет создавать, просматривать и удалять задачи.

Планируемые доработки:
Реализовать корректную перераспределяемую нумерацию задач
(при удалении задачи порядок должен автоматически пересчитываться).
Расширить модель задачи дополнительными полями
(например: описание, приоритет, срок выполнения, статус).
Добавить простой веб-интерфейс для работы через браузер
без использования Postman.

Проект находится в разработке.
